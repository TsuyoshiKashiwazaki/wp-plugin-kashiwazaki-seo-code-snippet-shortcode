# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-25

### Added
- Initial release
- Code snippet management (PHP, HTML, CSS, JavaScript)
- Shortcode support (`[kscss_snippet id="123"]` or `[kscss_snippet name="slug"]`)
- Gutenberg block integration
- Classic editor integration (TinyMCE button)
- Usage tracking for snippets
- Revision management for code changes
- Auto-insert feature: automatically insert snippets at specific positions in post content
- Multiple insertion position options (before/after content, first/last paragraph, H2/H3 headings)
- Multiple post type selection with checkboxes
- Check all / Uncheck all buttons for post type selection
- Accurate position detection that excludes content added by other plugins
- Support for content containing emojis
