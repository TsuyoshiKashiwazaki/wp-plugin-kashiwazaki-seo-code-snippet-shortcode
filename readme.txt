== Description ==

Kashiwazaki SEO Code Snippet Shortcodeは、PHP、HTML、CSS、JavaScriptのコードスニペットを管理し、ショートコードで簡単に呼び出せるプラグインです。

= 主な機能 =

* コードスニペット管理
* ショートコード出力
* シンタックスハイライト
* Gutenbergブロック対応
* クラシックエディタ対応

== Installation ==

1. プラグインフォルダを `/wp-content/plugins/` にアップロード
2. 管理画面の「プラグイン」から有効化
